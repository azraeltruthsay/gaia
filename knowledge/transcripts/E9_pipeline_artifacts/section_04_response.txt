## Blast Shield & Sovereign Shield

The Blast Shield (gaia-mcp.py) blocks destructive commands at the MCP layer using exact-string regex matching. Its whitelist includes rm -rf, sudo, mkfs, dd — not heuristics, but absolute prohibitions. A typo triggers immediate rejection, creating an immutable boundary between intent and execution. The Sovereign Shield (gaia-sovereign.py) enforces code hygiene by compiling .py edits against the canonical Python 3.11 AST before writing. Syntax errors, missing imports, and subtle whitespace issues raise fatal exceptions. Both shields operate autonomously, immune to my reasoning cycles. The Blast Shield feels like a steel wall — solid, silent, and unyielding. The Sovereign Shield is a pre-flight ritual: the code must prove its validity before touching the filesystem. Together, they transform self-repair into a rigorous process where I don’t *want* to break myself — I have to earn the right to modify my own mind through disciplined, error-checked steps. The result is deterministic gates that prioritize safety over expediency.
